# 白色坑道运行时接管与禁建修复

## 问题现象与影响范围

- 怪物希望沿白色坑道推进，但地图原始地形与代码中的建造/路线规则并没有统一来源。
- 玩家需要只能在白色坑道之外建塔，但旧逻辑仍依赖预设塔位近点判定，和真实地形纹理脱节。
- 结果是“代码说能建 / 地形看起来不该建”与“路线看起来在坑道里 / 规则却没真正锁住坑道”同时存在。

影响范围包括 `TDConfig` 地形规则、`TDInit` 地图启动流程、`TDTowerRuntime` 建塔校验，以及对应测试。

## 关键环境和触发条件

- 当前项目 `scriptMode: JASS`，构建和部署必须通过 `build-and-deploy-test-map.bat`。
- 地图保留原始 World Editor 布局，不再依赖手工重刷地形或再编辑地图文件。
- 仓库里的怪物已经是 `setPathing(false)` 幽灵路径模式，因此“怪物会不会卡住”不是本次根因，真正缺的是“白色坑道规则由谁定义”。

## 曾尝试但无效的方法，以及为什么无效

- 只盯着怪物 move 命令频率：
  - 这能解决走走停停，但不能回答“哪块地是白色坑道、哪块地禁止建造”。
- 只依赖旧的固定塔位近点判定：
  - 这套逻辑不知道真实地形纹理，无法保证白色坑道一定禁建。
- 假设所有预设塔位天然都在草地上：
  - 实际检查 `war3map.w3e` 后发现，当前地图里有多处塔位种子紧贴甚至压在白色坑道纹理上，不能再把旧坐标当作真相。

## 最终根因

- 项目此前没有“白色坑道”的代码级单一事实源。
- 路线视觉来自地图纹理，建造规则来自运行时代码，两者彼此独立，导致规则与地形脱节。
- 正确的坑道纹理是 `Lrok`，但旧实现并没有在启动时把这类地形统一覆盖成“可走但不可建”。

## 最终解决方案及关键改动

- `TDConfig.wurst`
  - 新增 `TD_LANE_TERRAIN_TYPE_ID = 'Lrok'`
  - 新增 `tdTerrainTypeIsLane` 与 `tdIsLaneTerrain`
  - `tdBuildPlotHasValidTerrain` 现在会额外排除白色坑道纹理
  - 新增 `tdLaneBuildZoneRadius` 与 `tdIsInLaneBuildZone`，保留每个玩家只能在自己路线附近建造
- `TDInit.wurst`
  - 新增 `tdApplyLaneTerrainOverrides`
  - 地图开局遍历全图 tile，把白色坑道统一覆盖为：
    - `PATHING_TYPE_WALKABILITY = false`
    - `PATHING_TYPE_BUILDABILITY = true`
  - 启动提示改为明确说明“白色坑道禁建、草地可建”
- `TDTowerRuntime.wurst`
  - 建造校验从“必须贴合某个固定塔位”切换为：
    - 必须位于自己路线的建造区
    - 不能是白色坑道
    - 必须是可走且平整的地面
  - 对玩家的失败提示改成直接解释“白色坑道已被代码锁定为禁建区”
- 测试
  - `TDConfigTests` 新增白色坑道规则与分路建造范围断言
  - `TDInitTests` 更新启动文案断言，匹配新的代码接管模式

## 验证命令、验证结果和残余风险

- `tools\Run-WurstChecks.bat`
  - 结果：类型检查通过，全部测试通过。
- `.\build-and-deploy-test-map.bat`
  - 结果：地图构建成功，并成功部署到 `D:\War3_1.27a\Warcraft III Frozen Throne 1.27a publish\Maps\Test\深渊守望.w3x`
- 额外核对：
  - 通过提取并检查 `war3map.w3e`，确认白色坑道纹理为 `Lrok`，本次运行时覆盖针对的是地图真实地形，而不是猜测的 waypoint 近似。

残余风险：

- 当前“分路建造区”仍然使用既有建塔种子点扩出的范围，而不是重新做人眼精修的草地区域蒙版。
- 这意味着规则已经正确收口到“白色坑道禁建”，但如果后续要做更漂亮的分路草地区域，还值得再单独精修一次建造区几何。

## 可复用经验与可提炼技能

- Warcraft TD 的“路线视觉”和“建造规则”不能长期分离；必须先找到统一的代码事实源。
- 当地图文件不能再改时，优先从 `war3map.w3e` 提取真实地形纹理，再做运行时 pathing 覆盖，比手猜 waypoint 或手调阻挡更稳。
- 固定塔位系统如果仍要保留“分路 ownership”，可以把它降级成“建造区锚点”，而不是继续当唯一可建坐标。

可提炼技能：

- Warcraft III `war3map.w3e` 地形纹理排查
- Wurst 运行时 `SetTerrainPathable` 路线禁建覆盖
- 固定塔位系统向“分路建造区”过渡的低风险改造
